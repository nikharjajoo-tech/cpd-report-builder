# Roadmap

## Phase 1: MVP
SharePoint ingestion, core RAG chapter generation, and basic citation grounding, validated against a single engagement.

## Phase 2
AI validation agent, version history, and the reviewer approval workflow.

## Phase 3
AI-assisted editing, the prompt benchmarking loop, and the admin ingestion monitoring dashboard.

## Future vision
Multi-engagement analytics, automated formatting templates, and expanded ingestion limits beyond 300MB.

## Sequencing logic
Ingestion and generation had to work reliably before anything else mattered, so Phase 1 proved the core loop against a single engagement before adding the quality and workflow layers. The validation agent came before AI-assisted editing deliberately: there was no point building a smoother editing experience for drafts that hadn't yet cleared a consistency bar.
