# About these assets

`hero.svg`, `dashboard-mockup.svg`, `editor-mockup.svg`, and `workflow.svg` are illustrative concept mockups built to represent the product for this portfolio, not screenshots of the live client system. The underlying platform was built for a confidential environmental consulting client, so real screenshots of it are not available to publish here.

If you have redacted or generalized screenshots of the actual interface you're able to share, swap them in under the same filenames and the README will pick them up automatically.
